8.3 Recurrent Neural Networks Some useful links: The Keras Functional API Recurrent layers with Keras. Recurrent Neural Networks (RNNs) are a class of neural networks designed to process sequential data by using internal memory to remember previous inputs (where order matters). They are used in various applications, especially in natural language processing (sentences, text, language modeling) and time series (stock prices, weather) analysis. Unlike feedforward networks, RNNs have memory: they pass hidden states from one step to the next. Types of RNN Layers (Keras) Keras provides several RNN variants: Layer Strength Weakness Use Case SimpleRNN Easy, fast Vanishing gradient problem Short sequences LSTM Long-term memory Slower, more parameters Long dependencies GRU Faster than LSTM Slightly less expressive Faster alternative Bidirectional Uses past + future context Doubles computation Context both directions Table 8.1: Comparison of RNN Layer Types • Long Short-Term Memory (LSTM) networks: Use ”gates” to control the flow of informa- tion, allowing them to remember context over much longer sequences. • Gated Recurrent Units (GRU): A simpler, more efficient variant of LSTMs that also addresses the vanishing gradient problem. • Bidirectional LSTM (BiLSTM) : a layer that processes a sequence in both forward and backward directions, allowing it to use information from past and future time steps simul- taneously. This improves performance on tasks where context matters, such as natural language processing. Lab 8 Deep learning 1 - Recurrent Neural Networks Learning objectives for this week are: 1. how to build a model using Tensorflow and Keras 2. to understand how Recurrent Neural Networks work 3. to be able to assess their performance and expressiveness 8.1 Getting started While machine learning uses simpler concepts, deep learning (a subset of machine learning) works with artificial neural networks (ANN), which are designed to imitate how humans think and learn. The term ‘deep learning’ is coined because the neural networks have various layers that enable learning, and re-learning. In the same way that the human brain is made up of neurons, neural networks are built up of layers of nodes. Individual layer nodes are linked to nodes in neighboring layers. The number of layers in the network indicates how deeper it is. In the human brain, a single neuron gets hundreds of impulses from other neurons. In an artificial neural network, signals move between nodes by applying weights to them. A heavier weighted node will have a greater impact on the next layer of nodes. The weighted inputs are compiled in the last layer to produce an output. Because deep learning systems process a vast quantity of data and perform multiple difficult mathematical calculations, they demand strong hardware. Deep learning training computa- tions, even with such sophisticated technology, can take a long time. Widely-adopted deep learning frameworks include: • PyTorch (highly flexible/research), • TensorFlow (production-ready/scalable), • Keras (beginner-friendly API). • JAX (high-performance). 44
